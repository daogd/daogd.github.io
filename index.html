网站构建
2025/10/18

前言

B站上有很多种“零成本搭建网站”，但要么是过时了，要么就是广告。但功夫不负有心人，我终于在上面找到了一个勉强可以成功的办法！详情见零成本搭建个人博客

很多朋友在看完漫长的主题部分后发现Gitee寄了，就直接阴暗扭曲的爬到up主脸上发癫了

在我五年级时我也是这么想的，然后为这玩意花了一晚上也没研究透，还被老爹肘击。

今年初一，我用github用的多了，渐渐发现了方法。

教程

废话不多说，直接开始！

  1. 首先先按主播的步骤在本地部署完成，然后获取密钥，点击github的设置，找到ssh密钥那一栏，点创建，输密钥，名字随便取

  2. 在本地博客文件夹bash一下，依次输入以下两个代码

    git config --global user.email "你的正确邮箱@gmail.com"
git config --global user.name "你的GitHub用户名"

  3. config文件最下面也要改一下，改成

      type: git
  repo: git@github.com:用户名/用户名.github.io.git
  branch: main

维护

需要写文章或什么就hexo new “文章名”，然后三件套

hexo cl
hexo g
hexo d
